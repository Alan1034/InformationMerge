# InformationMerge

一个集成了点击复制功能的展示组件 <br/>

import InformationMerge from "information-merge";

示例:

    const data = [
       {
         value: supplierAddr,
         label: "供应商地址",
       },
       {
         value: supplierAddrPoi,
         label: "供应商地址poi",
       },
    ];
    return <InformationMerge data={data} />;
